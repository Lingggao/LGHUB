# 我们对 Windows 质量的承诺

> 作者：Pavan Davuluri 与 Windows 预览体验计划团队  
> 译者：Ling Gao  
> 日期：2026 年 3 月 20 日

各位 Windows 预览体验成员，大家好，

我想以一名工程师的身份，和大家说说心里话。我的整个职业生涯都致力于打造人们每天都离不开的技术。Windows 影响着全球无数人的生活，几乎没有其他任何技术能出其右。每一天，我们都在倾听社区中大家使用 Windows 的真实感受。过去几个月，我和团队投入了大量时间仔细分析大家的反馈。通过这些反馈，我们真切地听到了大家对 Windows 的真情实感，以及希望它变得更好的期待。

今天，我将向大家分享我们的回应与行动。以下是我们将在本月及四月的 Windows 预览版中向各位预览体验成员展示的首批改进。

<img src="https://github.com/Lingggao/LGHUB/blob/main/Images/02_Vertical-Taskbar-1024x621.png?raw=true" width = "100%" />

> 桌面展示了位于屏幕底部、顶部、左侧和右侧的任务栏

- **更丰富的任务栏自定义选项（包括垂直和置顶）**：调整任务栏位置是大家呼声最高的需求之一。我们将引入把任务栏移至屏幕顶部或两侧的功能，让您能更轻松地打造个性化工作区。

- **聚焦核心，将 AI 融入最能发挥其价值的地方**：大家将会看到，我们在决定 Copilot 融入 Windows 的方式和位置时将变得更加克制且有针对性，把重心放在那些真正实用且精心打磨的体验上。为此，我们正在精简不必要的 Copilot 入口，这一改变将从截图工具、照片、小组件和记事本等组件开始。

- **减少 Windows 更新带来的干扰**：获取更新的过程应当可预期且易于安排，因此我们将把更多控制权交还用户。现在，您可以在设备初始化阶段跳过更新以便更快进入桌面；可以选择重启或关机但不安装更新；可以在需要时暂停更新更长时间。同时，我们还会减少自动重启和系统通知的频率，进一步降低更新对您的干扰。

- **速度更快、更可靠的文件资源管理器**：文件资源管理器是 Windows 中使用频率最高的界面之一。我们的首轮改进将聚焦提升启动速度、减少界面闪烁、实现更流畅的导航，并为日常文件操作提供更稳定可靠的性能保障。

- **对小组件和资讯流拥有更多掌控权**：小组件的存在价值应该是提供有用的信息，而不是让人分心或感到信息过载。我们将引入更安静的默认设置，让您能更好地控制小组件的显示时机与方式，并进一步优化 “发现” 资讯流的个性化推荐。

- **更简洁、透明的 Windows 预览体验计划**：预览体验计划是大家参与塑造 Windows 的未来的重要途径，因此，它的预期目标和参与方式应该更加清晰易懂。我们正在做出调整以提升大家的参与体验，包括明确各频道的定义、简化新功能的获取方式、提供更高质量的版本、让大家更直观地看到反馈如何改进 Windows 以及创造更多与我们直接交流的机会。

- **全新升级的反馈中心（即日起上线）**：大家的声音是改进 Windows 的关键，提交反馈、查看他人的想法应当是一件轻松简单的事。今天，我们向各位预览体验成员推送了反馈中心有史以来最大规模的更新。重新设计的界面与体验将让提交反馈和参与社区互动的过程变得更加快捷、顺畅。

<img src="https://github.com/Lingggao/LGHUB/blob/main/Images/03_FeedbackHub2.0-1024x624.png?raw=true" width = "100%" />

> 全新的反馈中心应用，展示了重新设计的主界面和反馈表单

在此基础上，我们设定了更长远的计划和今年的核心目标，力求全面提升 Windows 11 的质量水准。各项工作正紧锣密鼓地推进。在今年余下时间发布的预览版中，大家将真切地体验到这些看得见摸得着的进展。

昨晚，我有幸在西雅图与一小部分 Windows 预览体验成员进行了座谈。我倾听了大家的想法、解答了疑问并进一步分享了我们未来的发展方向。西雅图见面会只是一个开始，接下来我们的团队将前往全球更多城市，与各地的 Windows 社区开展面对面的交流。

感谢大家一直以来用高标准鞭策我们。Windows 属于我们，更属于你们。我们承诺不断夯实其基础，并在大家真正需要的地方带来有价值的创新。

请继续向我们提出宝贵的反馈意见，让我们携手共创 Windows 的未来。

祝好，

## 交付性能、可靠性与匠心

以下是我们今年提升 Windows 11 质量的计划，重点关注**性能**、**可靠性**和**精心打造的体验**。这些领域对您的 Windows 使用体验有着重大影响：启动和响应速度、在真实工作负载下的稳定性，以及整体体验的一致性与细腻程度。

<img src="https://github.com/Lingggao/LGHUB/blob/main/Images/05_K2-Visual-1024x279.png?raw=true" width = "100%" />

### [性能]

我们致力于提高 Windows 11 的响应速度和一致性，让性能表现更流畅、更可靠。

今年，我们将持续优化系统性能、应用响应速度、文件资源管理器以及适用于 Linux 的 Windows 子系统（WSL），确保 Windows 在您切换应用与工作负载时保持流畅。

**提升系统性能**：减少 Windows 占用的资源，为您的操作释放更多性能空间。

- 更快速、响应更灵敏的 Windows 体验，早期的改进已缩短文件资源管理器等组件的启动时间

- 提升内存效率，降低 Windows 的基准内存占用，为运行应用腾出更多容量

- 更一致的性能表现，即便在高负载下也能确保应用全天候灵敏响应

**更流畅、响应更快的应用交互**：通过将 Windows 核心体验迁移至 WinUI 3 框架，降低交互延迟。

- 优化 Windows 体验所依赖的共享 UI 基础设施，在平台层面降低交互延迟与开销

- 通过将更多体验迁移至 WinUI 3，提升开始菜单等 Windows 核心体验的响应速度

**优化文件资源管理器基础体验**：降低搜索、导航和文件操作的延迟，提高可靠性。

- 大幅降低搜索、导航、右键菜单的延迟

- 大文件的复制与移动将更快速、可靠

- 提升常用文件任务的启动速度和响应能力

**改进适用于 Linux 的 Windows 子系统（WSL）体验**：为在 Windows 上使用 Linux 工具与环境的开发者提高性能、可靠性和集成度。

- 提高 Linux 与 Windows 之间的文件传输性能

- 优化网络兼容性与吞吐量

- 更简洁的初始化设置和入门体验

- 通过更强的策略控制、安全性与治理能力实现更好的企业级管理

### [可靠性]

可靠性是信任的基石。在最需要的时候，您的设备应当随时就绪并正常运转。

在整个操作系统中，我们将重点提升 Windows 预览体验计划、驱动程序与应用、更新以及 Windows Hello 等方面的基准可靠性。

**增强 Windows 预览体验计划的可靠性与质量**：明确各 Insider 频道的预期，提高版本质量门槛并强化反馈功能，以便在广泛发布前提高版本质量。

- 更清晰地展示每个预览版包含的功能，让您心中有数

- 更自主地控制体验哪些新功能、更轻松地切换频道，以适配对稳定性或早期体验的需求

- 进入各频道的版本质量更高，在发布前进行更严格的验证和反馈评估

- 强化 Windows 的反馈闭环，确保问题能够更快地被识别、排序、解决

**提高系统、驱动程序和应用的可靠性**：通过加强系统稳定性、驱动质量与应用可靠性，在我们活跃的芯片、软件供应商（ISV）和原始设备制造商（OEM）生态中提供更平稳、可靠的 Windows 11 体验。重点包括：

- 减少系统级崩溃、提升驱动质量和应用稳定性以巩固 Windows，确保电脑随时流畅可靠

- 建立更轻松、快速、稳定的蓝牙配件连接，减少 USB 相关的崩溃与连接中断，提升打印机的发现速度和连接性能

- 更可靠的摄像头与音频连接，提升工作和娱乐效率

- 更一致的设备唤醒（包括进一步提升扩展坞场景下的唤醒一致性），助您快速回到工作状态

**优化 Windows 更新体验**：更新更快速、更可预测，并且能更清晰地控制重启和时间。

- 减少 Windows 更新带来的干扰，降低强制重启的频率并减少中断。将设备重启频次向每月一次靠拢，但需要更快获取新功能与修复补丁的组织和用户仍可照常更新

- 更直接地控制更新，包括根据需要长时间暂停更新，以及在不强制安装更新的情况下重启或关机

- 更快速、可靠的更新体验，在更新过程中提供更清晰的进度提示，内置恢复机制以便在出现问题时保持设备稳定

**改进 Windows Hello 生物识别身份验证**：我们正在强化 Windows Hello 登录功能，使其更可靠、从容且安全，在减少摩擦的同时增强设备准确识别您的可靠性。

- 更可靠的面部识别，确保登录精准有效

- 更快速、可靠的指纹登录，减少重试次数

- 在 ROG Ally X 等掌机上实现更轻松的安全登录，在设置过程中及菜单内提供完整的手柄支持以便创建 PIN

### [匠心]

对我们来说，“匠心” 是通过易用性、打磨、连贯性和精细化将功能性产品转变为受人喜爱的产品的准则。

今年，您将看到我们致力于提升整体体验的易用性，提供更多个性化功能，在系统中减少噪音、干扰并增加控制力。这包括深思熟虑如何以及在何处将 AI 引入 Windows，并以透明度、选择权和控制权为先，确保新功能是增强体验而非使其复杂化。

**优化开始菜单和任务栏体验**：让 Windows 核心界面更可靠、更灵活、更个性化，让您以最适合自己的方式操作电脑。

- 开始菜单和任务栏将提供更一致、可靠的应用和文件访问，让内容切换全天候流畅

- 扩展任务栏个性化选项，包括备选任务栏位置和更小的任务栏，让您更好地控制界面以适应工作流

- 开始菜单中更相关的 “推荐” 板块将展示您最关心的应用和内容，提供清晰的控件来定制体验或直接关闭

**更专注的用户体验，更少的干扰**：让 Windows 体验更安静，助您保持专注、减少分心。

- 新的 Windows 设备的设置过程更安静、精简，页面数量和重启次数更少，入门更简单

- 小组件默认以更有针对性的方式展示信息，内容一目了然，减少不必要的打扰

- 更简单的设置方式让您可以轻松地根据偏好个性化、加入或关闭小组件与信息流

- 减少通知提醒，助您随时保持专注

**增强搜索功能**：在 Windows 的各个界面提供更快速、更准确的结果与一致的搜索体验。

- 更快找到重要内容，搜索功能将清晰呈现应用、文件和设置，让您快速获得正确结果

- 搜索结果更清晰、可信，设备本地内容与网页结果明确区分，易于理解

- 在任务栏、开始菜单、文件资源管理器和设置中提供一致的搜索体验

作为这项工作的一部分，我们正在改进 Windows 的底层架构以提高质量，并根据您的反馈在最关键的地方进行创新。

这也包括向 Windows 预览体验成员发布新体验前，在真实硬件和使用场景下进行更深入的验证和更广泛的测试，并对新功能的引入位置和方式采取更审慎的态度。其目标是让您获得更高质量的版本、更有意义的创新，并在选择尝试哪些功能时拥有更高的灵活性。这就是未来我们持续构建并交付 Windows 11 的方式，我们将月复一月、更有信心地提供更好的体验。

配合微软的 “[安全未来倡议（SFI）](https://www.microsoft.com/en-us/trust-center/security/secure-future-initiative)”，我们将随各个版本持续增强 Windows 的安全性，内置新的功能并加强默认安全性，以保护用户、设备和数据。

在我们不断改进和创新的过程中，期待您持续提供反馈，告诉我们可以在哪些方面让 Windows 变得更好。
