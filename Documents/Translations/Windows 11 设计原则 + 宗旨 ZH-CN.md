**Windows 11 设计原则 + 宗旨**

**—— 引领我们为人工智能时代设计 Windows 的指导力量**

原文：[**Windows 11 design principles + tenets**](https://microsoft.design/articles/windows-11-design-principles-tenets) (Microsoft Design)

> **作者** | Diego Baca (微软合伙人、设计总监)
> **译者** | Ling Gao (微软 Windows 商业 & 云冠军)

原文撰于 2025 年 10 月 8 日，10 月 13 日译。

房子和家究竟有何不同？房子是物理结构 —— 是梁柱、墙壁和玻璃的组合，将形式与功能融为一体。而家则承载情感，是内心感受与居所意义的交织。Windows 正是二者的结合。作为微软 Windows 的设计者，我们不仅搭建了一座能够容纳 15 亿人信息的 “房子”，更赋予每个人真正融入其中的能力。我们的传承始于对实用性与普及性的追求。在计算机尚未普及的 20 世纪 80 至 90 年代，我们为 PC 引入了个性化。从那时起，Windows 逐渐演变为一个连接人们、帮助他们蓬勃发展的家园。

Windows 承载着世界领袖、知识工作者、创意思考者和 CEO 们的办公空间。那扇大门就是标志性的 “开始” 菜单，通往陈列着各类应用与文件 “房间” 的客厅。这里是他们的项目、日程和沟通的家园。这里孕育创业火花，构筑宏伟事业，并在 Microsoft 365 应用中安放着他们的梦想。对客户而言，Windows 是专属的私人空间，一个可以随手把钥匙扔到桌上的地方。因此，我们始终致力于精心设计一座真正称得上 “家” 的房子。

我们在瞬息万变的社会和技术环境中践行这一理念，而其基石正是所有用户体验领域共同遵循的设计原则和宗旨。然而，随着时代更迭，昔日建房的宗旨未必适用于今日的新居。因此，为满足客户不断变化的需求，我们重新定义了 Windows 11 的设计原则。新原则注重简洁、统一与匠心，旨在带来熟悉感、沉稳感和即刻可用的体验。在人们迈入人工智能时代之际，本文将深入剖析塑造我们设计理念的原则 + 宗旨。

有意识创新意味着在信任和客户反馈的基础上持续改进 Windows —— 仅在确能提升用户体验时才作出调整，让人们能在熟悉且舒适的环境中探索全新的 AI 驱动设计。

### **有意识创新**

有意识创新意味着以信任、价值和客户成功为导向。为此，我们会以恰当的节奏为客户带来新的设计理念。为在舒适与变革之间取得平衡，我们依靠紧密的迭代循环与实时客户反馈。哪些方案行之有效？哪些理念引发共鸣？哪些功能被广泛使用？哪些设计打破了用户习惯？回顾 Windows 的发展历程，有一点始终未变 —— Windows 是人们完成任务的关键工具。任何阻碍高效完成任务的设计都应被舍弃，唯有能显著提升体验的变革才值得推进。

Windows 8 的发布深刻印证了这一点。当我们推出与前代截然不同的 UI (包括移除 “开始” 按钮) 时，人们陷入了迷茫。这个他们使用多年的核心工具突然变得陌生。快进到如今的人工智能时代，一个以快速变革和集体不确定性为特征的时代 —— 与客户共创比以往任何时候都更加重要。邀请他们加入我们的设计旅程，正是我们拥抱客户的方式。例如，若能在熟悉的环境中尝试全新的 AI 优先设计模式，人们会感到更安心。这不仅给予人们信心，也提供了一个安全的空间，让他们探索适合自身工作流的解决方案。通过有意识创新，我们得以在合适的节奏下突破边界。

在 Windows 中，匠心意味着从整体构思到细微之处都倾注心血。通过融合深思熟虑的设计、一致的体验和灵活的系统，我们让用户感受到宾至如归，同时支持创新，在微软生态中实现高效生产力。

### **匠心 + 可扩展系统**

匠心是精巧，是风格，更是我们表达关怀的承诺。匠心不仅关乎宏观的端到端体验，也体现在对每个微观细节的斟酌。对 Windows 而言，匠心就是一切。我们的核心宗旨之一是：以统一性铸就信心 —— 一次学会，处处可用。正因如此，只有明确哪些是不可动摇的基石，哪些是可以优化的部分，才能在不干扰用户体验的前提下持续改进。Windows 应当清晰、沉稳，在 “不打扰” 中助力客户达成目标。自然流畅的交互能延续使用习惯，串联不同场景。而维系客户信任的唯一途径，就是保持透明和尊重，让客户始终掌控全局。

随着设计语言不断演进，我们致力于做出深思熟虑的调整，让 Windows 更实用、更易用、也更具吸引力。自然的色彩赋予它层次感，鲜明的活力激发创造力，跃动的人文气息扑面而来。新增的柔性材质和圆角设计营造出舒适的体验。为更好地融入广阔的微软生态，我们采用了最新版 Fluent 设计系统。我们希望赋予人们更深邃的时间感与空间感，激发节奏和流动感，因此将引入更多动画和过渡效果。纵观微软丰富的产品矩阵，我们正着力构建产品间的关联纽带，支持人们喜欢的工作方式，在统一体系下打造多元体验。

Windows 建立在灵活互联的基础之上 —— 将设计与技术深度融合，以支持全新的 AI 赋能体验。随着 Windows 持续进化，我们重新构想了 Microsoft 画图、PowerPoint 等经典应用，让日常工作更轻松，同时保留人们信赖的熟悉外观与体验。

要做好这件事，需要构建恰当的基础架构：既要高度互联互通，统一各类要素，又要足够灵活，适应时代发展，从而支持人工智能的新概念与新模式。随着 Windows 不断演进，其深厚的创新积淀与人工智能浪潮交汇，为我们重新构想 Microsoft 画图、PowerPoint 等经典应用带来了独特机遇。借助 AI 驱动的工具，Windows 能更轻松地实时搜索并启动操作项。但问题在于，如何在拓展这些新体验的同时，不破坏平台的行为逻辑与视觉一致性？

我们采用可识别的设计模式、控件和组件。几十年来，数以百万计的用户通过学习使用 Windows 提高了工作效率。他们养成了长期的使用习惯，这些习惯反过来也塑造了我们服务客户、引导他们体验新功能的方式。如今，数十亿用户已将 Copilot、Microsoft 365 等产品视为深度融入其工作流的工具。正因如此，我们采用统一的产品开发模式，以实现规模化与敏捷性。通过跨团队、跨领域的紧密协作，我们持续创造、重塑、改进并扩展产品。

Windows 为每位用户提供自适应体验。其核心在于个性化，让人们能根据自身独特需求塑造 Windows，从而释放全新可能。

### **为所有人量身定制**

过去，定制客户体验意味着对每个选项进行硬编码的标准化处理。如今，千篇一律的方案已无法满足所有人。我们并非为 15 亿人打造完全相同的产品，而是赋予他们自适应的体验。借助高度情境化的人工智能系统，人们可以自主选择符合自身需求的 Windows 体验。试想专为速度和沉浸感调校的掌上游戏版 Windows、专为全神贯注打造的高效生产力空间、专为无障碍操作设计的界面。这些都令我们无比振奋，因为它们释放了无限可能。个性化正是 AI 系统的核心宗旨，我们对其未来充满期待。

Windows 的设计安全、可信且合规，从设计之初就将隐私保护与安全防护融为一体，通过透明性与可控性建立信任。正如房子的灵魂成就了家，Windows 的设计宗旨正是守护人们最珍视的事物。

### **安全、可信且合规的设计**

通过言行一致建立信任，意味着在每个产品决策中践行我们的宗旨。我们深知，人工智能与隐私保护是人们最关心的议题。因此，安全性并非事后补救，而是从设计之初就内嵌其中，合规性也深植于内核。从最初的迭代阶段到最终发布，我们都将安全措施作为 Windows 架构的基本组成部分加以整合。在不断扩展的数字世界中，数据保护如同房屋的警报系统，守护着其中的一切。

房子内部有其灵魂，是它唤起了 “家” 的感觉。没有灵魂，房子便不是家。至于 Windows，我们兼顾二者。
