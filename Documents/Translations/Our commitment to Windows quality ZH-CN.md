# 我们对 Windows 质量的承诺

> 原文：[***Our commitment to Windows quality***](https://blogs.windows.com/windows-insider/2026/03/20/our-commitment-to-windows-quality)  
> 作者：Pavan Davuluri 与 Windows 预览体验计划团队  
> 译者：Ling Gao  
> 日期：2026 年 3 月 20 日

各位 Windows 预览体验成员，大家好，

我想以工程师的身份，和大家说些心里话。我的整个职业生涯都致力于打造一个人们每天都离不开的技术平台。Windows 影响着全球无数人的生活，几乎没有其他技术能与之相比。每一天，我们都在倾听社区中大家使用 Windows 的真实声音。过去几个月，我和团队投入了大量时间，仔细梳理大家的反馈。通过这些反馈，我们真切地感受到了大家对 Windows 的深厚情感，以及希望它变得更好的殷切期待。

今天，我将向大家分享我们的回应与行动。以下是我们将在本月及四月的 Windows 预览版中，向各位预览体验成员推出的首批改进。

<img src="https://github.com/Lingggao/LGHUB/blob/main/Images/02_Vertical-Taskbar-1024x621.png?raw=true" width = "100%" />

> 桌面任务栏支持位于屏幕底部、顶部、左侧或右侧

- **更丰富的任务栏自定义选项（包括垂直和置顶）**：调整任务栏位置是呼声最高的需求之一。我们将引入把任务栏移至屏幕顶部或两侧的功能，助您更轻松地打造个性化工作区。

- **聚焦核心，将 AI 融入最能发挥价值的场景**：大家将会看到，在决定 Copilot 融入 Windows 的方式与位置时，我们将更加克制且有针对性，重心放在那些真正实用且精心打磨的体验上。为此，我们正在精简不必要的 Copilot 入口，这一改变将从截图工具、照片、小组件和记事本等应用开始。

- **减少 Windows 更新带来的干扰**：获取更新的过程应当可预期且易于安排，因此我们将把更多控制权交还给您。您可以在设备初始化阶段跳过更新，以便更快进入桌面；可以选择重启或关机而不安装更新；也可以在需要时暂停更新更长时间。同时，我们还会减少自动重启和系统通知的频率，进一步降低更新对您的干扰。

- **速度更快、更可靠的文件资源管理器**：文件资源管理器是 Windows 中使用频率最高的界面之一。我们的首轮改进将聚焦于提升启动速度、减少界面闪烁、实现更流畅的导航，并为日常文件操作提供更稳定可靠的性能保障。

- **对小组件和资讯流拥有更多掌控权**：小组件的价值在于提供有用信息，而非让人分心或感到信息过载。我们将引入更安静的默认设置，让您能更好地控制小组件的显示时机与方式，并进一步优化资讯流 “发现” 的个性化推荐。

- **更简洁、透明的 Windows 预览体验计划**：预览体验计划是大家参与塑造 Windows 未来的重要途径，因此其目标和参与方式应更加清晰易懂。我们正在做出调整以提升大家的参与体验，包括明确各频道的定义、简化新功能的获取方式、提供更高质量的版本、让大家更直观地看到反馈如何改进 Windows，以及创造更多与我们直接交流的机会。

- **全新升级的反馈中心（即日起上线）**：大家的声音是改进 Windows 的关键，提交反馈、查看他人想法应当轻松简单。今天，我们向各位预览体验成员推送了反馈中心有史以来最大规模的更新。重新设计的界面与体验，将使提交反馈和参与社区互动的过程更加快捷、顺畅。

<img src="https://github.com/Lingggao/LGHUB/blob/main/Images/03_FeedbackHub2.0-1024x624.png?raw=true" width = "100%" />

> 全新的反馈中心应用，展示了重新设计的主界面和反馈表单

在此基础上，我们制定了更长远的规划与今年的核心目标，力求全面提升 Windows 11 的质量水准。各项工作正紧锣密鼓地推进。在今年余下时间发布的预览版中，大家将切实体验到这些看得见、摸得着的进展。

昨晚，我有幸在西雅图与部分 Windows 预览体验成员进行了座谈。我倾听了大家的想法，解答了疑问，并进一步分享了未来的发展方向。西雅图见面会只是一个开始，接下来，我们的团队将前往全球更多城市，与各地的 Windows 社区开展面对面交流。

感谢大家一直以来以高标准鞭策我们。Windows 属于我们，更属于你们。我们承诺不断夯实其基础，并在大家真正需要的地方带来有价值的创新。

请继续向我们提出宝贵意见，让我们携手共创 Windows 的未来。

## 交付性能、可靠性与匠心

以下是我们今年提升 Windows 11 质量的计划，重点聚焦于**性能**、**可靠性**和**精心打磨的体验**。这些领域对您的 Windows 使用体验至关重要：它们直接关系到启动与响应速度、真实工作负载下的稳定性，以及整体体验的一致性与细腻度。

<img src="https://github.com/Lingggao/LGHUB/blob/main/Images/05_K2-Visual-1024x279.png?raw=true" width = "100%" />

### [性能]

我们致力于提升 Windows 11 的响应速度与一致性，让性能表现更加流畅、可靠。

今年，我们将持续优化系统性能、应用响应速度、文件资源管理器以及适用于 Linux 的 Windows 子系统（WSL），确保您在切换应用与处理不同工作负载时，都能享受流畅体验。

**提升系统性能**：减少 Windows 自身资源占用，为您的操作释放更多性能空间。

- 更快速、响应更灵敏的 Windows 体验，早期的改进已显著缩短了文件资源管理器等核心组件的启动时间

- 提升内存效率，降低 Windows 的基准内存占用，为运行您的应用程序腾出更多容量

- 更一致的性能表现，即便在高负载环境下，也能确保应用全天候保持灵敏响应

**更流畅、响应更快的应用交互**：通过将 Windows 核心体验迁移至 WinUI 3 框架，有效降低交互延迟。

- 优化共享 UI 基础设施，在平台层面降低交互延迟与系统开销，优化所有依赖此基础的 Windows 体验

- 通过将开始菜单等更多核心体验迁移至 WinUI 3，显著提升其响应速度

**优化文件资源管理器基础体验**：降低搜索、导航和文件操作的延迟，提高整体可靠性。

- 显著减少搜索、导航及右键菜单的响应延迟

- 大文件的复制与移动将变得更加快速且稳定可靠

- 提高日常文件任务的启动速度与响应能力

**改进适用于 Linux 的 Windows 子系统（WSL）体验**：为在 Windows 上使用 Linux 工具与环境的开发者提供更高的性能、可靠性和集成度。

- 优化 Linux 与 Windows 之间的文件互传速度

- 改善网络兼容性与吞吐量

- 更简洁的入门体验，简化初始化设置流程

- 通过更强的策略控制、安全性与治理能力，实现更高效的企业级管理

### [可靠性]

可靠性是信任的基石。在最需要的时候，您的设备应当随时就绪并正常运转。

在整个操作系统中，我们将重点提升 Windows 预览体验计划、驱动程序与应用、系统更新以及 Windows Hello 等方面的基准可靠性。

**增强 Windows 预览体验计划的可靠性与质量**：明确各 Insider 频道的预期，提高版本质量门槛并强化反馈功能，以便在广泛发布前显著提升版本质量。

- 清晰展示每个预览版包含的具体功能，让您心中有数

- 让您能更自主地选择体验哪些新功能，并更轻松地切换频道，以适配您对稳定性或早期尝鲜的不同需求

- 进入各频道的版本将经过更严格的验证和反馈评估，确保高质量交付

- 优化反馈机制，确保问题能够被更快地识别、排序并解决

**提高系统、驱动程序和应用的可靠性**：通过加强系统稳定性、驱动质量与应用可靠性，在我们活跃的芯片、软件供应商（ISV）和原始设备制造商（OEM）生态中，提供更平稳、可靠的 Windows 11 体验。重点包括：

- 减少系统级崩溃，提升驱动质量和应用稳定性，确保电脑随时流畅可靠

- 建立更轻松、快速、稳定的蓝牙配件连接，减少 USB 相关的崩溃与连接中断，提升打印机的发现速度和连接性能

- 确保摄像头与音频连接更可靠，提升工作和娱乐效率

- 实现更一致的设备唤醒体验（包括进一步提升扩展坞场景下的唤醒一致性），助您快速回到工作状态

**优化 Windows 更新体验**：让更新更快速、更可预测，并提供更清晰的重启和时间控制权。

- 减少干扰，降低强制重启的频率并减少中断，目标是将设备重启频次向每月一次靠拢，但需要更快获取新功能与修复补丁的组织和个人用户仍可照常更新

- 更直接的控制，包括根据需要长时间暂停更新，以及在不强制安装更新的情况下选择重启或关机

- 更快速可靠的体验，在更新过程中提供更清晰的进度提示，并内置恢复机制，以便在出现问题时保持设备稳定

**改进 Windows Hello 生物识别身份验证**：我们正在强化 Windows Hello 登录功能，使其更可靠、从容且安全，在减少摩擦的同时增强设备准确识别您的能力。

- 更可靠的面部识别，确保登录精准有效

- 更快速可靠的指纹登录，减少重试次数

- 在 ROG Ally X 等掌机上实现更轻松的安全登录，在设置过程中及菜单内提供完整的手柄支持以便创建 PIN

### [匠心]

对我们而言，“匠心” 是将功能性产品转化为受人喜爱产品的核心准则。它体现在易用性、精细打磨、体验连贯性以及无微不至的细节优化中。

今年，我们将致力于提升整体体验的易用性，提供更丰富的个性化功能，在系统中减少噪音与干扰，并赋予用户更强的控制力。这包括深思熟虑地将 AI 引入 Windows，始终将透明度、选择权和控制权放在首位，确保新功能是为了增强而非复杂化您的体验。

**优化开始菜单和任务栏体验**：让 Windows 的核心界面更可靠、更灵活、更个性化，让您能以最适合自己的方式操作电脑。

- 开始菜单和任务栏将提供更稳定、一致的应用与文件访问体验，确保内容切换全天候流畅无阻

- 增加任务栏的自定义能力，包括支持调整任务栏位置以及提供更紧凑的任务栏尺寸，让您更好地掌控界面以适应独特的工作流

- 开始菜单中的 “推荐” 区域将展示与您最相关的应用和内容，并提供清晰的控件，让您轻松定制或直接关闭该功能

**更专注的用户体验，更少的干扰**：打造更安静的 Windows 体验，助您保持专注，减少分心。

- 新 Windows 设备的初始设置过程将更加安静、精简，减少页面数量和重启次数，让入门体验更简单顺畅

- 小组件默认将以更精准的方式展示信息，内容一目了然，减少不必要的打扰

- 提供简便的设置方式，让您轻松根据偏好个性化、开启或关闭小组件与信息流

- 优化通知管理，帮助您随时保持专注状态

**增强搜索功能**：在 Windows 的各个界面提供更快速、更准确的结果与一致的搜索体验。

- 更快找到重要内容，搜索功能将清晰呈现应用、文件和设置，助您迅速获得正确结果

- 更清晰可信的结果，明确区分设备本地内容与网页搜索结果，让信息来源易于理解且可信

- 在任务栏、开始菜单、文件资源管理器和设置中提供统一、连贯的搜索交互体验

作为这项工作的一部分，我们正在改进 Windows 的底层架构以提高质量，并根据您的反馈在最关键的地方进行创新。

这也意味着，在向 Windows 预览体验成员发布新体验之前，我们将在真实硬件和多样化使用场景下进行更深入的验证与更广泛的测试。同时，我们将对新功能的引入位置和方式采取更加审慎的态度。我们的目标是为您提供更高质量的版本、更有意义的创新，并让您在选择尝试哪些功能时拥有更高的灵活性。这就是未来我们持续构建并交付 Windows 11 的方式 —— 月复一月，更有信心地提供更好的体验。

配合微软的 “[安全未来倡议](https://www.microsoft.com/en-us/trust-center/security/secure-future-initiative)（Secure Future Initiative, SFI）”，我们将随各个版本持续增强 Windows 的安全性。通过内置新功能和加强默认安全设置，全方位保护用户、设备和数据的安全。

在我们不断改进和创新的过程中，期待您的持续反馈。请告诉我们，您认为还有哪些方面可以让 Windows 变得更好。
