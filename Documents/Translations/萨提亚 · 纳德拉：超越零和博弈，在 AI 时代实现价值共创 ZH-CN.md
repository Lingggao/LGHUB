# 萨提亚 · 纳德拉：超越零和博弈，在 AI 时代实现价值共创

**编者按**

原文：https://x.com/satyanadella/status/1989755076353921404

**作者：Satya Nadella（微软公司董事长兼首席执行官）**

我一直在思考人工智能（AI）平台浪潮所带来的净收益究竟是什么。真正的问题在于，如何赋能每一家企业，使其充分把握平台转型的机遇，构建自身的原生 AI 能力与企业价值（而不是在无意中让自己的独特价值被转移到科技行业！）。

比尔 · 盖茨（Bill Gates）曾说过一句名言：只有一个平台的所有使用者创造的经济价值超过创建该平台的公司自身价值时，它才真正成为一个平台。这就是 “正和博弈” 未来的核心所在。

即便在我们这个略带 “零和” 思维的行业，依然可以建立为所有参与方创造价值的合作伙伴关系。我们与 OpenAI 的合作就是一个很好的例子。我们的投资帮助他们实现了规模化发展，而他们的研究也加速了我们的创新进程。健康的平台与合作伙伴关系正是如此 —— 能够催生并加速进步。

我们本周宣布的成果就是最好的证明。全球首个 AI 超级工厂由微软与 OpenAI 共同设计，融合了我们在前沿模型训练与推理领域打造的三代 AI 超级计算机的经验。这一成果也得益于与英伟达的深度合作，三家公司从模型架构到芯片微架构在全栈层面协同优化，共同推动了技术进步！

我们还助力 AMD 加入 GPT 模型推理阵营，帮助其快速提升自身的 AI 软件栈能力。

如今，这些基础设施已具备可扩展性，能够支持从初创公司到大型企业的所有组织自主开展模型训练与推理。

编程领域也呈现出同样的趋势。得益于 AI，编程工具这一类别正在不断壮大，未来可能成为规模最大的软件类别之一。我从未见过有分析师问过 Visual Studio 的营收是多少！而现在，大家都对 AI 编程工具充满期待。这正是 “正和” 效应的又一体现 —— 当整个类别被重新定义，市场蛋糕就能做大十倍！我们通过 GitHub Copilot 争取市场份额，同时也通过 GitHub 和 Agent HQ 为他人提供平台。

当然，这个时代的真正考验，并不在于哪家科技公司打破了估值纪录，而在于经济和社会能否整体迈向全新的高度。

当制药公司利用 AI 进行模拟，仅用一年就将新疗法推向市场，而非十二年；当制造商借助 AI 在一夜之间重塑供应链；当教师能为每个学生量身定制个性化课程；当农民可以预测并防止作物歉收 —— 那时，我们才能确信这套体系正在发挥作用。

让我们超越 “零和” 思维与 “赢家通吃” 的炒作，专注于构建最广泛的能力，让每家企业都能驾驭技术的力量，在本地取得成功，进而带动更广泛的经济增长与社会福祉。要确保每家企业都能掌握自身的命运与自主权，而不仅仅是在新闻稿中宣布与某家科技 / AI 公司达成合作 —— 更糟的是，在看似 “合作” 的关系中，陷入长期被抽取价值的陷阱。

我们深知，互联网浪潮曾为世界带来巨大的正面影响，但本地媒体等行业也因此遭遇了空心化冲击。这一次，我们有机会确保这项技术得到广泛普及，赋予各方选择权与控制权，从而实现真正的互利共赢。

归根结底，这场新的技术浪潮不仅为我们提供了更大的想象空间，也设定了更高的集体成就目标。我们每个人都应发挥自己的力量！
