# Brandon LeBlanc 调任新职，告别 Windows 预览体验计划团队

**作者** | Ling Gao - 微软 Windows 商业 & 云冠军

当地时间 2025 年 9 月 19 日，Windows 预览体验计划负责人 Amanda Langowski 通过社交媒体 X 宣布，微软首席项目经理 Brandon LeBlanc 不再负责该团队相关事务，未来将在 Windows 团队承担新的工作职责。

Amanda 在推文中表示：“Brandon 永远是 Windows Insider 的一员，我们衷心感谢他为项目注入的极客精神、透明沟通与社区热忱。祝贺他迈向职业生涯新阶段。”

Brandon LeBlanc 于 2008 年加入微软，2015 年起投身 Windows 预览体验计划，负责撰写预览版博客文章、与 Insider 成员沟通及收集反馈等工作。今年 3 月，他晋升为微软首席项目经理，直至此次岗位调整。
