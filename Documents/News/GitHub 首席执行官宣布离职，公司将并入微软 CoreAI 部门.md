# GitHub 首席执行官宣布离职，公司将并入微软 CoreAI 部门

**作者** | Ling Gao (高楷修) - 微软 Management 社区冠军

**2025 年 8 月 11 日，代码托管平台 GitHub 首席执行官 (CEO) Thomas Dohmke 宣布将卸任现职，回归创业。与此同时，GitHub 及其领导团队将整体并入母公司微软的 CoreAI 部门。**

Thomas Dohmke 将继续留任至 2025 年底，以确保平稳过渡。他自 2021 年 11 月起担任 GitHub 首席执行官，任期将满四年。本次调整的更多细节安排将在晚些时候公布。

GitHub 成立于 2008 年，是全球开发者广泛使用的代码构建、存储、管理和共享平台，目前拥有超 10 亿个代码库与分支，注册开发者人数逾 1.5 亿。2018 年 6 月，微软宣布收购 GitHub，交易于同年 10 月完成。此后，GitHub 作为微软子公司独立运营至今。

此次接收 GitHub 的微软 CoreAI 部门组建于 2025 年 1 月，全称为 “CoreAI - Platform and Tools (平台与工具) 工程部门”。该部门整合了微软内部的开发部门、人工智能 (AI) 平台以及首席技术官 (CTO) 办公室中的多个核心团队，致力于构建端到端的 Copilot 与 AI 应用技术栈，加速推进微软战略路线图的实施。目前，CoreAI 部门由微软执行副总裁 Jay Parikh 领导。
