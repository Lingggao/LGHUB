## Microsoft 宣布开源 “适用于 Linux 的 Windows 子系统” (WSL)

**编辑** | Ling Gao

2025 年 5 月 19 日，Microsoft 宣布开源 “适用于 Linux 的 Windows 子系统” (WSL)，并欢迎社区开发者做出贡献，共同将 Windows 塑造成为跨平台应用开发的最佳操作系统。
