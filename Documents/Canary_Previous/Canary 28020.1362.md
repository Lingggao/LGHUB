# WIP Canary 28020.1362 更新简报

> [!TIP]
>
> &emsp;Ling Gao 制 (2025/12/19)，不代表 Microsoft 公司立场、态度。
>
> &emsp;鼓励阅读官方原文：[英文更新公告](https://blogs.windows.com/windows-insider/2025/12/15/announcing-windows-11-insider-preview-build-28020-1362-canary-channel)、[中文更新公告]() (暂无)。

## 一、新功能

1. 全屏体验 (FSE) 面向更多 Windows 11 掌机提供。

<img src="Images/28020.1362_1.png" width = "50%" />

2. Windows 工作室效果支持更多摄像头硬件。

<img src="Images/28020.1362_2.png" width = "45%" />

3. 更多文件资源管理器窗口支持深色模式。
	- 复制、移动、删除对话框
	- 进度条、图表视图
	- 确认操作 (跳过、覆盖、选择) 对话框
	- 二次确认对话框
	- 错误对话框

<img src="Images/28020.1362_3.png" width = "40%" />

<img src="Images/28020.1362_4.png" width = "40%" />

<img src="Images/28020.1362_5.png" width = "40%" />

<img src="Images/28020.1362_6.png" width = "40%" />

4. 高级菜单新增 “虚拟工作区” 选项。

## 二、变更和改进

1. 优化了 “单击以执行” 功能。
2. 优化了设置菜单中的 Agent。
3. 优化了 “拖拽托盘” 功能。
4. 大幅优化了文件资源管理器。
5. 优化了 “蓝牙和其他设备” 设置菜单。
6. 优化了 “Windows 聚焦” 功能。
7. 将更多键盘选项从控制面板移动至设置。
8. 改善了 HID 标准键盘的背光性能。
9. 支持触觉反馈的触控笔将在与系统界面交互时提供振动反馈。
10. 更新了 OneDrive 图标。
11. 优化了 “快速计算机恢复 (QMR)” 功能。
12. 优化了 Windows 搜索界面。
13. 改善了解锁设备后加载任务栏的表现。

## 三、修复 & 已知问题

详见[官方原文](https://blogs.windows.com/windows-insider/2025/12/15/announcing-windows-11-insider-preview-build-28020-1362-canary-channel)和[灵糕中心](https://github.com/Lingggao/LGHUB)。

部分重要修复：

1. 修复了 “文件资源管理器不显示 EXIF 文件缩略图” 的问题。
2. 修复了 “文件资源管理器出现旧版工具栏” 的问题。
3. 修复了 “文件资源管理器自定义视图意外恢复默认” 的问题。
4. 修复了 “文件资源管理器上下文菜单无响应” 的问题。
5. 修复了 “文件资源管理器解压大文件时出现灾难性错误” 的问题。
6. 修复了 “文件资源管理器首页无响应” 的问题。
7. 修复了 “网络和 Internet 菜单无响应” 的问题。
8. 修复了 “隐藏任务栏选项被自动关闭” 的问题。
9. 修复了 “点击任务栏窗口预览时，窗口无法回到前景” 的问题。
10. 修复了 “从待机状态恢复时无法联网” 的问题。
11. 修复了 “部分游戏提示检测到不支持的显卡” 的问题。
12. 修复了 “应用和浏览器在滚动鼠标时短时间无响应” 的问题。
13. 修复了 “首次登录新账户时异常缓慢” 的问题。
14. 修复了 “幻灯片锁屏背景出现内存泄漏” 的问题。
15. 修复了 “旁白功能阅读时随机卡住” 的问题。
16. 修复了 “更新并关机时无法成功关机” 的问题。
17. 修复了 “更新 Windows 时出现 0x8007042B 错误” 的问题。
18. 修复了 “关闭任务管理器后进程一直在后台运行” 的问题。
19. 修复了 “部分应用拉起 ‘打开 / 保存对话框’ 时无响应” 的问题。
20. 修复了 “任务视图界面意外自动弹出” 的问题。
